# DeepAgentForce 学习与开发日志

本目录用于自动记录对 DeepAgentForce 项目的学习与开发过程。

## 📁 目录结构

- [`00-project-overview.md`](./00-project-overview.md) - 项目整体概览与架构
- [`01-dev-log.md`](./01-dev-log.md) - 日常开发记录
- [`02-learning-notes.md`](./02-learning-notes.md) - 学习笔记与知识点
- [`03-decisions.md`](./03-decisions.md) - 技术决策与设计思路
- [`04-troubleshooting.md`](./04-troubleshooting.md) - 问题排查与解决方案
- [`template-dev-session.md`](./template-dev-session.md) - 开发记录模板
- [`template-learning.md`](./template-learning.md) - 学习笔记模板

## 🚀 使用方法

### 开始新的开发会话
```bash
# 复制模板
cp learn-docs/template-dev-session.md learn-docs/01-dev-log-$(date +%Y%m%d).md
```

### 记录学习内容
```bash
# 复制模板
cp learn-docs/template-learning.md learn-docs/02-learning-$(date +%Y%m%d-%H%M).md
```

## 📝 自动记录

本目录下的文档会在每次开发会话后自动更新，记录：
- Git 提交历史
- 文件变更摘要
- 新增的功能模块
- 遇到的问题与解决方案
